= 关于封面

本书封面上的动物是一只黄颊山雀（Machlophus spilonotus）。这种鸟生活在阔叶混交山林以及东南亚的公园和花园中。

黄颊山雀亮黄色的脸颊和颈背与黑色的鸟冠、喉部和前胸形成了鲜明的对比，这使其很容易辨认。封面上的这只雄鸟身体呈灰色，黑色翅膀上布满白色斑点和条纹。雌鸟则身体呈橄榄色，翅膀上有淡黄色条纹。

黄颊山雀以小型无脊椎动物、蜘蛛、水果和浆果为食，在森林的中低层觅食。像北美山雀、山雀和山雀家族中的其他鸟类一样，黄颊山雀以短而起伏的方式飞行，伴随着翅膀快速摆动。

虽然黄颊山雀的生存状态被列为最不受关注的，但 O'Reilly 图书封面上的许多动物濒临灭绝，它们对世界很重要。
